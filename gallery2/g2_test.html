Modèles du plugin Gallery 2

Consultez le code source de ce fichier pour avoir les versions en balises # MODELE pour insertion dans vos squelettes

tests modele gphoto:

test 1:

3 photos (items Gallery 14, 20 et 27) séparées par un HR, taille 200px, pas de lien dessus:
<gphoto|item=14:20:27|sep=<hr/>|taille=200|lien=non>
[(#MODELE{gphoto}{item=14:20:27}{sep=
}{taille=200}{lien=non})]

test 2:

les 2 dernières photos publiées dans Gallery, flottantes à droite:
<gphoto|dernieres=2|right>
[(#MODELE{gphoto}{dernieres=2}{align=right})]

test3:

une photo au hazard avec le lien sur l'image à la place de Gallery
<gphoto|lien=img>
[(#MODELE{gphoto}{lien=img})]
