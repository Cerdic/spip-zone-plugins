[(#REM) icone=images/tdm.png] [(#REM) titre= Table des Matières] [(#REM) descriptif= Les paramètres configurables de la Table des Matières sont les suivants :
  • la longueur définit le nombre de caractères maximum que consituera chaque ancres,
  • le séparateur définit le caractère qui s'intercalera entre chaque mot pour remplacer les espaces,
  • et le nombre d'intertitres minimum définit le nombre d'intertitres d'un texte à partir duquel une table des matières sera affichée.
