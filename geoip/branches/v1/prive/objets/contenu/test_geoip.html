Dans les squelettes de SPIP

Récupérer le code pays d'après une adresse IP

[(#GET{ip}|geoip_informations{geoip_country_name_by_addr})]

[(#GET{domaine}|geoip_informations{geoip_country_code_by_name})]
#BOITE_FERMER #BOITE_OUVRIR{'Aide mémoire - PHP','info'}

Dans les fichiers PHP

Récupérer le code pays d'après une adresse IP

geoip_informations($ip);

Récupérer le nom du pays d'après une adresse IP

geoip_informations($ip, 'geoip_country_code_by_name');
#BOITE_FERMER #BOITE_OUVRIR{'Documentation','info'}
  • geoip_country_code_by_name - Affiche les deux lettres du code pays
  • geoip_country_name_by_name - Affiche le nom complet du pays
