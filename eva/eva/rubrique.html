Créer une (sous-)rubrique


Attention : seuls les administrateurs ont le droit de créer des rubriques et sous-rubriques.
  1. Entrez dans l'interface d'administration
  2. Allez dans :
  3. Naviguez dans l'arborescence, en descendant dans les rubriques jusqu'à ce que vous arriviez dans la rubrique où vous souhaitez créer votre sous-rubrique. Ou à la racine du site pour créer une rubrique.
  4. Pour créer une (sous-) rubrique, cliquez sur :
  5. Dans la page d'édition de la rubrique remplissez le titre (éventuellement le descriptif et/ou le texte explicatif)
  6. Si vous vous êtes trompé de niveau, vous pouvez directement mettre votre rubrique dans une autre, avec :

    Sélectionnez la rubrique dans laquelle vous voulez placer votre rubrique (ou racine du site). C'est aussi par ce même procédé que l'on peut déplacer (plus tard) une rubrique avec tout ce qu'elle contient (sous-rubrique, articles, ...)

  7. Lorsque votre rubrique est saisie, cliquez sur [Valider] (en bas de page)
Si une rubrique est "vide" (ne contient aucun article ou site référencé) elle n'apparaîtra pas dans le site.
