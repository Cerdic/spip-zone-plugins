# OwlCarousel

## SOURCES :

http://www.owlgraphic.com/owlcarousel/#more-demos

## Utilisation


### Afficher le portfolio d'un article

```
<INCLURE{fond=noisettes/owlcarousel-article-portfolio}
		{id_article=28}
		{id-carousel=sommaire_hero}
		{navigation=false}
		{autoplay=true}
		{autoplayHoverPause=true}
		{lazyLoad=true}
		{env}
		{ajax} />
```
