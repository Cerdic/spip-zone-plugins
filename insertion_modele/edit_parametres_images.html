Explications

Largeur : Correspond à la taille de l’image lorsque le texte est aligné avec l’image.[ La taille par défaut est de (#CONFIG{inserer_modeles/largeur_defaut})px (pixels)]
Précisions : Dans le cas d’une image en taille maximum, ce n’est pas pris en compte ([(#CONFIG{inserer_modeles/largeur_max})px]).

Hauteur : Correspond à la hauteur de l’image. Peut être utilisé avec les 3 type d’insertions d’images.

Zoom : Permet d’activer un effet de zoom sur le clic d’une image (si elle est trop petite, le désactiver)

Titre / Description : Ajoute le texte et le titre de l’image sous la photo.
