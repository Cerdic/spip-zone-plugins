Outil de prototypage de sites sous SPIP

Pour la gestion de :

- pages 
	- id_page
	- nom_page
	- objectif
	- particularités
	- DA (image)
	- url
	- commentaires

- blocs
	- id_bloc
	- id_page (ou id_bloc... pour le cas de bloc inclus dans un bloc)
	- nom_bloc


- liens
	- id_lien
	- id_parent (peut etre un bloc)
	- url
