[(#CACHE{0})][(#REM) Copyright Daniel FAIVRE 2007-2011 Modele ACS-Cat pour la balise pour insérer un lecteur audio (mp3) ou video dans un article pour le document joint. Dépend des composants Audio et Video Exemples: Si le document docnn est de format mp3, affiche la vignette si elle est personnalisée. Si le document docnn est une vidéo dans un format lisible, affiche le lecteur et un lien de téléchargement sans vignette. Si une vignette au format jpeg non progressif existe, elle sert d'image de lancement du player. Affiche le TITRE, le DESCRIPTIF suivi du lecteur flash permettant de faire jouer ce fichier. Sinon, utiliser le comportement normal d'une balise doc dans un article. Pour les lecteurs audios, la liste des mp3 est celle des liens contenus dans le parent du composant player Permet l'édition (sécurisée) des titres et descriptifs des documents depuis le site public si le plugin crayons est installé. Paramètres - parameters: w=largeur (ne s'applique qu'au lecteur video) - width (video only) h=hauteur (ne s'applique qu'au lecteur video) - height (video only) nodownload=y: pas de lien de telechargement - no download link (audio & video) small=y: lecteur audio minimal (bouton play/pause seul) ][(#SET{is_video,Flash Video|MPEG4|OGV|WEBM})]
