# Stocks

Gestion de quantité.

## BOUCLES

```html
<BOUCLE_kelStock(STOCKS){objet ?}{id_objet ?}>
	Stock : #QUANTITE
</BOUCLE_kelStock>
```

## BALISES

`#QUANTITE` affiche la quantité dans un contexte sans utiliser de boucle
