Plugin d'exemples d'utilisation de CFG

Ce plugin permet de découvrir des fonctionnalités de CFG pour SPIP 2.0

Cette partie (formulaires publics) permet de montrer le fonctionnement des nouveaux formulaires dynamiques dits 'CVT' (Charger, Verifier, Traiter) introduits dans SPIP 2.0.
Cette nouvelle gestion permet de créer des formulaires publics ou privés, en ajax ou non assez facilement.

Ils peuvent (avec le plugin CFG) être couplés à celui ci. Ainsi, un formulaire CVT ne possédant pas de fonction de chargement utilisera automatiquement le parseur et les fonctions de CFG pour charger, vérifier et traiter le formulaire.

Note : La balise #CFG_ARBO{x} permet de lister les resultats de #CONFIG{x}
