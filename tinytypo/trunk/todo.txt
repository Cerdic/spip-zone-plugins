== WHY
- Ce plugin a pour vocation essentielle de modifier certains sélecteurs CSS du code généré par SPIP afin de bénéficier de TOUS les styles de Tiny Typo.
- Accessoirement, il distribue aussi les styles de Tiny Typo, mais ce n’est pas le plus important (faire un plugin juste pour distribuer du CSS… bof).

Mode d’emploi préalable de Tiny Typo pour SPIP :
https://contrib.spip.net/Installer-Tiny-Typo-dans-SPIP?var_mode=preview
