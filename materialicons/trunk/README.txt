Plugin Material Icons
pour utiliser facilement les icônes Material Icons (Pixsellz) en rédaction ou programmation
v0.1.0 dev - welcome contributions !

les modèles dispo :
pour les icones liées à des articles
modeles/article_material_icone.html
<article425|material_icone|largeur=200px|hauteur=200px|center|couleur=red|couleur2=orange>

pour mes icônes liées à des mots-clefs
modele/mot_material_icone.html
<mot2|material_icone|largeur=200px|hauteur=200px|couleur2=orange>

pour ionsérer n'importe où dans les textes
modeles/material_icone.html
<material_icone|style=Rounded|categorie=Communication|icone=phone|couleur=purple|center|largeur=200px|hauteur=200px>
<material_icone|style=Two Tone|categorie=Communication|icone=mobile_screen_share|center|largeur=60px|hauteur=60px|couleur2=orange>
