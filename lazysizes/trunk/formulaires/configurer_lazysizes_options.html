LazySize Options

Options de configuration

<:lazysizes:cfg_explication_css:>

(lazyload !default) Marqueur pour les éléments qui bénéficierons du chargement différé

(lazypreload !default) Marqueur pour les éléments qui bénéficierons du chargement différé sur l'événement onload (chargement de la page)

(lazyloading !default) Classe ajouté dès le début du chargement. Peut être utilisée pour signaler la progression.

(data-src !default) Attribut utilisé comme source du fichier.

(data-sizes !default) Attribut utilisé pour l'attribut sizes.

(data-srcset !default) Attribut utilisé pour l'attribut srcset.
