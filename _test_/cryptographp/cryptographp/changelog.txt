Version 1.3


Ajouts:
-------
- Actualisation (facultative) du cryptogramme "à la volée" sans recharger la page.
- Lancement du générateur de nb aléatoire indispensable sur anciennes versions de PHP
- Qualification: PHP >= 4.2.0 (PHP >= 4.3.2 conseillé)



Modifications:
--------------
- Simplification reconnaissance fichier de config
- Utilisation de la variable session 'configfile' pour indiquer le fichier de configuration.
- Modification de l'algorythme de recadrage horizontal: 4 fois plus rapide que le précédent
- Simplification de l'installation et de la verification des cryptogrammes
- Gestion de la casse obligatoire ou non.
- Remplacement des commandes imagepng(imagecreatefrompng()) par readfile()
- Renommage cryptographp.include.php => cryptographp.inc.php
- En mode "cypteasy", le code ne commence plus nécessairement par une consonne.
- Changement de la police par défaut meilleur compatibilité avec PHP 4.2.0 à 4.3.1



Corrections:
------------
- Compatibilité avec les sessions dont le nom <> "PHPSESSID"
- Compatibilité avec les sites error_reporting bavards (ex: EasyPHP)
- utilisation fonction imagecolorallocate si imagecolorallocatealpha n'existe pas (PHP<4.3.2)
- utilisation fonction imagefilter si elle existe (PHP >= 5.0.0)
- Traitement des polices pour compatibilité toutes versions de GD (noms avec ou pas ".ttf")



Suppressions:
-------------
- suppression d'une portion de code deja en commentaire dans la version précédente.



Notes:
------

Cryptographp est compatible avec PHP >= 4.2.0
Certaines fonctionalités nécessitent des versions plus récentes de PHP et de la bibliothèque graphique GD.

Généralités
Le format GIF n'est pas géré par la librairie "GD <2.0.18"
=> Utilisez de préférence le format PNG bien meilleur que GIF et qui offre également la transparence.

Fonctions compatibles uniquement si PHP >= 4.3.2
- Transparence des caractères

Fonctions compatibles uniquement si PHP >= 5.0.0
- Transformation de l'image en niveau de gris
- Brouillage de l'image par méthode Gauss
