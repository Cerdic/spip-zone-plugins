Options générales

Les noms d'options correspondent à ceux de jsMath.

Message si fontes manquantes (option showFontWarnings)
Autoriser le mode global (option allowGlobal)

Chemin d'accès

Entrez ci-dessous le chemin d'accès au répertoire jsMath. Si ce champ est laissé vide, le plugin cherchera ce répertoire en utilisant les fonctions de SPIP.

C'est sans doute une mauvaise idée de remplir ce champ si jsMath est visible par SPIP, par contre en cas de mutualisation ça peut servir ... Dans tous les cas, il doit s'agir d'une adresse dans le même domaine que les pages fabriquées par SPIP.
