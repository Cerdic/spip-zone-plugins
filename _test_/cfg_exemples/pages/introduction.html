Plugin d'exemple de la balise #FORMULAIRE_CFG

Ce plugin permet de découvrir des fonctionnalités introduites dans la version 1.4 de CFG, la balise #FORMULAIRE_CFG.

Le menu '<:navigation:>' permet de naviguer dans les differents exemples.

La balise #CFG_ARBO{x} permet de lister les resultats de #CONFIG{x}
