----------------------------------------------------------------------
Permet l'utilisation du plugin Agenda dans le squelettes Alternatives.
----------------------------------------------------------------------

Dans le squelette, quatre fichiers à modifier : 
	themes/alternatives/sommaire.html, 
	themes/alternatives/article.html, 
	themes/alternatives/inc/inc-menu.html
	themes/alternatives/inc/inc-menu-principal.html

----------------------------------------------------------------------
Pour sommaire.html, placer le code : 
	[(#MODELE{encart_agenda}{lang}{self=#SELF})] 

	après la ligne : 
	<INCLURE{fond=inc/inc-annonces}{lang}>

Pour inc/inc-menu-principal.html, placer le code : 
	<B_agenda><li id="menu-principal-contact"><a href="<BOUCLE_agenda(EVENEMENTS){0,1}>[(#URL_PAGE{evenements})]</BOUCLE_agenda>" title="<:icone_agenda:>"><:icone_agenda:></a></li></B_agenda>
	
	après la ligne qui se termine par : 
	<:accueil_site:></a></li>

Pour inc/inc-menu.html, placer le code : 
	[(#MODELE{rubrique_calendrier}{self=#SELF}{var_date=archives}{date=#ENV{archives,#DATE{lang}{id_rubrique})]
	
	après la ligne : 
	<div class="menu" id="menu-rubriques">

Pour article.html, placer le code : 
	[(#MODELE{evenements_article}{id_article}{lang}{self=#SELF})] 
	
	après la ligne  : 
	<INCLURE{fond=inc-petition}{id_article}{self=#SELF}>
	
----------------------------------------------------------------------
Pour vous simplifer la vie, les fichiers modifiés ainsi modifiés ont été placés dans : 
	plugins/agenda_alternatives/modifications
	
Il suffit donc pour vous de remplacer les 4 fichiers originaux dans :
	themes/alternatives et themes/alternatives/inc

----------------------------------------------------------------------
Vous obtiendrez donc :

	- Un mini calendrier dans le cadre de gauche
	- Un encart 'Agenda' dans le cadre de droite avec les prochains évènements
	- Un bouton 'Agenda' supplémentaire en haut, menant à :
	- Une page 'evenements' qui liste les évènements (du jour/du mois, prochains, passés)
	- La liste des évènements associés au bas de chaque article
