    pubban_commande   [line 27]

    void pubban_commande( [ $panier = '']  )

    PRINCIPE DE FONCTIONNEMENT

    Les données récupérées depuis le formulaire public (page 'spip.php?page=adds' ou copie personnalisée - utilisation du formulaire 'form_adds') sont systématiquement envoyées vers la fonction de ce fichier :

    1. pubban_commande($panier)

    où $panier est le panier généré résumant la commande :

    1. pub:titre_emplacement:nombre_d_affichages:prix_par_affichage:id_pub

    La fonction doit retourner TRUE ou FALSE selon l'état de la commande :

      • FALSE sinon,
      ou rediriger tout simplement vers une nouvelle page.

      A noter que lors de sa création (enregistrement en base de données), la publicité porte le statut '0cree'. Pour être présentée dans les bannières du site, elle doit être activée par l'appel :

      1. pubban_editer_pub($id_pub, array("statut"=>"2actif"))

      Vous pouvez également lui donner le statut '1inactif' si son activation est prévue à une date précise.
