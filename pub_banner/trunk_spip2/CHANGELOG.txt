## Pub Banner - NOTE D'INTENTION ##

PubBanner est un gestionnaire de bannières publicitaires sous forme de plugin pour SPIP 
(ici en version 2.0 et plus). L'ensemble du code devra être correctement commenté par utilisation
de DocBlocks pour génération d'une doc via le plugin PhpDoc.
