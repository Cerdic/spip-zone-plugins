Le thème "Griseus" pour SPIP 3 est divisé en 3 plugins, auxquels correspondent les sous-répertoires :

- icones : jeu d'icônes pour l'espace privé
- vignettes : vignettes des types de fichiers du plugin 'medias' -> espace prive + public
- icones_barre : icônes de la barre d'édition du plugin 'porte plume' -> espace privé + public
- theme : thème pour l'espace privé (à venir)
