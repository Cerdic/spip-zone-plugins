-----------------------------------------
	PRÉSENTATION
-----------------------------------------
	Export au format tableur CSV des articles pour publication papier
	Extrait certaines données saisies dans SPIP à des fins de publication papier et/ou de publipostage
	Ce plugin est axé sur les articles SPIP. Il exporte au choix de l’utilisateur les éléments les composants (titre, sous-titre, texte, p.s...) ainsi que :
    * leur rubrique (et ses éléments la composant)
    * les mots-clés associés (titre et nom du groupe) :
          o aux rubriques
          o aux articles
    * les pétitions
    
    voir https://contrib.spip.net/Export-CSV-des-articles-pour

-----------------------------------------
	INSTALLATION :
-----------------------------------------
	0 - LE PLUGIN CFG DOIT ÊTRE PRÉALABLEMENT INSTALLÉ
	
	1 - Comme pour tous les plugins SPIP, décompacter le contenu de l'archive dans le dossier "plugins" situé à la racine du site.
	
	2 - Dans l'interface privée du site, bouton "Configuration" -> "Gestion des plugins", activer le plugin "Export CSV"
	
	3 - Ouvrir la page du plugin en survolant le bouton "Édition" puis en  cliquant sur l'icône "Extraire les données". Sur cette page, s'affiche :
		- un aperçu des données extraites,
		- un lien vers CFG pour configurer les données à extraire,
		- un lien pour télécharger les données au format CSV.
		
	4 - Configurer le plugin via CFG
