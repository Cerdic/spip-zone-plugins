spip_d3js
============

Intégrer d3.js dans SPIP

## Librairies fournies

* [D3.js](https://github.com/mbostock/d3/)
* [d3-legend](https://github.com/susielu/d3-legend/)
* [d3pie](https://github.com/benkeen/d3pie/)
* [topojson](https://github.com/mbostock/topojson)
