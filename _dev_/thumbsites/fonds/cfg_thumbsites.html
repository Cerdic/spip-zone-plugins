Activation du plugin

Thumbshots

Thumbshots propose un service gratuit qui ne nécessite pas d'identification.

Les vignettes générées seront de 120 x 90 pixels.

Thumbshots.de

Thumbshots.de propose un service équivalent à thumbshots.com. Bien que les services soient identiques, ce sont bien 2 sites distincts.

Les vignettes générées seront de 120 x 90 pixels.

Girafa

Pour utiliser ce service vous devez être inscrit sur le site girafa
Il existe une version gratuite si vous souhaitez moins de 2000 captures de site par jour.

Les vignettes générées seront au maximum de 160 x 120 pixels (se paramètre sur le site girafa)

Websnapr

Pour utiliser ce service vous devez être inscrit sur le site girafa
La version gratuite permet de traiter de 250.000 captures de site par mois, à un rythme de 80 captures par heure.
