ex: Pentium4 2.5Ghz, Pentium4 2.6Ghz, Pentium4 2.7Ghz etc...

* boutique_ristourne *
**********************
Table qui contient toutes les ristourne, promotion, remises etc... de la plateforme.
___
id_ristourne
cible => enum client, catégorie, gamme, produit
type => pourcent,brut
id_cible => id_client ou id_categorie ou id_produit du profuit "père" d'une gamme ou id_produit.
cummulable => oui, non
priorite => en %
