Titre, logo, URL du site

On va vous donner ici la démarche pour personnaliser le titre du site, son URL (par défaut) et le logo qui apparaît en haut à droite de la page d'accueil.

Le Logo du site

Ce logo correspond au logo du site référencé qui est dans la rubrique "Titre, logo, URL du site". Modifiez le logo du site référencé pour faire apparaître votre logo à gauche du nom de votre site.

Cliquez ici, pour accéder directement au site référencé.

Le Nom et URL du site

Pour les modifier, il faut le faire par l'interface d'administration du site : Administration du site > Configuration du site > Contenu du site, puis modifiez :

  • Nom de votre site : s'affichera sur la page d'accueil (par exemple : nom de l'établissement)
  • Adresse (URL) du site public : URL d'EVA-PingOO qui sera utilisée dans les liens vers la page d'accueil.

C'est d'ailleurs dans cette même partie, que vous pourrez configurer votre site (articles, brèves, forums, ...).

Cliquez ici, pour accéder directement à l'interface d'administration du site.

Remarque : vous trouverez juste en dessous "Adresse e-mail du webmestre"..
