Présentation :

EVA-WEB est un squelette complet doté de fonctionnalités telles que
des albums photographiques automatiques par mot-clefs ou un menu dépliant.
Consulter <http://spip-edu.edres74.net/rubrique.php3?id_rubrique=22> pour
plus de détails ainsi que <http://zone.spip.org/trac/spip-zone/wiki/LesProjets>
pour les autres projets issus d'EVA-WEB.

Le plugin eva est une reprise de ces squelettes, modifiés (parfois de manière
importante : le menu et le sommaire sont entièrement modifiés, les squelettes
sont prévus pour être utilisés avec le plugin crayons, etc.) et sous forme d'un
unique plugin autonome dont l'installation ne demande aucune modification d'un
SPIP déjà installé. Ainsi, le plugin est compatible avec SPIP 1.9.2 alors
qu'EVA-WEB ne l'est pas. La version plugin est utilisée en production à <http://www.college-petite-camargue.net/>.
