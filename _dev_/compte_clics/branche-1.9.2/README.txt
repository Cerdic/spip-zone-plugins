Origine
-------

Ce plugin a ete realisee a partir des contributions : http://www.spip-contrib.net/Compter-les-clics-sur-les-liens et http://www.plugandspip.com/spip.php?article37 .


Fonctionnalites
----------------

Pour le moment, ce plugin permet de compatibiliser les clics par site syndiqué et par article syndique. Dans ce dernier cas, tout clic sur un article incremente le compteur de l'article et le compteur du site.
En outre pour etre utilisable, une balise #TOTAL_CLICS a ete fournie pour les articles syndiques et les sites syndiques.


Installation / Desinstallation
------------------------------

Lors de l'activation du plugin, celui-ci rajoute les colonnes necessaires dans la base de donnees de SPIP pour y stocker les informations sur les clics. La desinstallation supprime ces colonnes et permet donc un retour a la situation anterieure sans aucune trace.

Utilisation
-----------

La page de demonstration fourni tous les exemples possibles :
- utilisation du filtre permettant de compter les clics
- boucles integrant la balise TOTAL_CLICS
- tri dans les boucles par nombre de clics
