+---------------------------------------+
| INSTALLATION                          |
+---------------------------------------+
1 - Comme pour tous les plugins Spip, décompacter le contenu de l'archive dans le dossier "Plugins" situé à la racine du site.
2 - Dans l'interface privée du site, valider le plugin "Recherche Multi-critères" (Onglet "Configuration" -> "Gestion des plugins").
3 - Ouvrir la page de configuration du plugin en cliquant sur l'icône "Recherche Multi-critères" (deux petites clés).
4 - Configurer le plugin.

+---------------------------------------+
| AFFICHAGE PUBLIC                    |
+---------------------------------------+
Pour une recherche sur tout le site (sans possibilité de reduire à une rubrique)
- Placer la balise [(#MODELE{recherchemulticriteres})] là ou vous voulez voir s'afficher le formulaire de recherche
Pour une recherche sur une rubrique (avec possibilité de reduire ou non la recherche à la rubrique)
- Placer la balise [(#MODELE{recherchemulticriteres}{id_rubrique})] dans la page rubrique.html (ou autre) là ou vous voulez voir s'afficher le formulaire de recherche

+---------------------------------------+
| DESCRIPTION                           |
+---------------------------------------+
Ce plugin permet d'effectuer une recherche d'articles en sélectionnant un ou plusieurs mots-clé. La recherche retournera la liste d'articles associés à ces mots-clé.
Deux types de recherche sont proposés :
1 - Effectuer une recherche sur tout le site en affichant le formulaire de recherche dans n'importe quelle page du site (ou sur une page dédiée)
2 - Effectuer une recherche en affichant le formulaire de recherche dans une page rubrique (ou autre permettant de récupérer l'id_rubrique)

Dans les deux cas, vous pouvez configurer les groupes de mots apparaissant dans le formulaire de recherche, ains que le nombre de colonnes pour l'affichage public.

Vous pouvez rechercher les articles associés à tous les mots ou non (et/ou)
Vous pouvez limiter la recherche par date de publication des articles (-7 jours, -1 mois, -3 mois, -12 mois, tous)
Dans le cas de l'affichage par rubriques, vous pouvez limiter ou non la recherche à la rubrique en question.
