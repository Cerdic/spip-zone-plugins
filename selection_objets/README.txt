Pour des raisons historiques, ce plugin s'appellait sélection d'objet.
Mais en réalité, il liait les objets.
Un nouveau plugin en a repris le code, en changeant simplement la terminologie, pour parler de liaison plutôt que de sélection.
Ce plugin ne doit donc plus bouger, les modifications devant se faire dans le nouveau plugin, liaison d'objet.
