Fonctionnalités du plugin '#POPUP'

Attention à ne pas abuser des liens ouvrant une nouvelle fenêtre, par respect pour les internautes ...

-  Liens hypertextes ouvrant une fenêtre externe

Pour inclure dans vos textes un lien s’ouvrant dans une nouvelle fenêtre (sous réserve que JavaScript soit installé et actif sur l’ordinateur de l’utilisateur), vous pouvez utiliser le raccourci typographique d’appel du modèle "popup".

Exemple :

affiche :

mon texte

-  Aperçus de documents ouvrant une fenêtre externe

Sur le même principe que pour le modèle ci-dessus, vous pouvez utiliser le raccourci typographique du modèle "popup_img" pour générer un aperçu d’un document SPIP dans une nouvelle fenêtre.
