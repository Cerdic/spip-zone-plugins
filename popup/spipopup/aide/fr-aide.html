Fonctionnalités du plugin '#POPUP'

Attention à ne pas abuser des liens ouvrant une nouvelle fenêtre, par respect des internautes ...

-  Liens hypertextes ouvrant une fenêtre externe

Pour inclure dans vos textes un lien s’ouvrant dans une nouvelle fenêtre (sous réserve que JavaScript soit installé et actif sur l’ordinateur de l’utilisateur), vous pouvez utiliser le raccourci typographique d’appel du modèle "popup".

Exemple :

affiche :

mon texte

N.B. Vous pouvez également préciser une URL complète au lien d’une référence d’objet SPIP (paramètre "lien=..."), le lien généré ouvrira alors cette URL.

-  Aperçus de documents ouvrant une fenêtre externe

Sur le même principe que ci-dessus, vous pouvez utiliser le raccourci typographique d’appel du modèle "popup_img" pour générer un aperçu d’un document SPIP de type image ouvrant un lien en nouvelle fenêtre.
