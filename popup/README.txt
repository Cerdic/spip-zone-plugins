Plugin "Popup" pour SPIP 2.0+
------------------------------------------

Ce plugin est proposé en deux versions distinctes :

- une version plugin complet et individuel (répertoire 'spipopup' de ce repository)

- une version lame pour le Couteau Suisse (répertoire 'popup_lame_cs' de ce repository) - lisez le fichier 'README.txt' de la lame pour installation.

------------------------------------------

Plugin Couteau Suisse : http://www.spip-contrib.net/Le-Couteau-Suisse
