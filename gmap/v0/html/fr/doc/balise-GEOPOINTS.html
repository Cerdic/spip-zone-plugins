Rôle

Cette balise permet de récupérer le nombre de points définis sur un élément et ses descendants.

C'est le pendant, en balise, de ce que fait la boucle GEOTEST.

Paramètres

NomDescriptionValeur
id_article/id_rubrique... Objet à partir duquel les points sont comptés. {id_article} ou {id_article=39}, {id_rubrique}, {id_document}, {id_breve}, {id_mot}, {id_auteur}
type Type du marqueur recherché sur un élément. Parmi les types définis dans le paramétrage du plugin.
recursif Drapeau indiquant si le comptage inclut ou non les descendants. oui / non.
