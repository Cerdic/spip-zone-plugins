Rôle

Cette balise permet de tester qu'une fonctionnalité est présente dans l'implémentation sélectionnée pour les cartes (voir le paramétrage du plugin).

Par exemple :

[(#GEOCAPABILITY{LayerAuto}|oui) ... ]

Permet d'ajouter du code si la fonction LayerAuto (correspondant par exemple à la couche Wikipedia disponible en Google Maps V2) est accessible.

Paramètres

La balise n'utilise qu'un paramètre qui n'est pas nommé.

Les valeurs suivantes sont actuellement utilisées :

  • markers : gestion des marqueurs (ajout, modification, suppression, affichage ou masquage).
  • infowindows : gestion des bulles d'information quand on clique sur un marqueur (GMap ne permet pas d'afficher des bulles d'information sans lien avec un marqueur).
  • layerkml : affichage des fichiers KML.
  • layerauto : affichage de couches "automatiques", comme la couche wikipedia de Google Maps V2 (c'est le seul exemple).
  • layeractions : possibilité d'afficher ou de masquer les couches KML par script.
  • geocoder : recherche par adresse depuis l'interface de géolocalisation.
  • dragmarkers : possibilité de déplacer les marqueurs de l'interface de géolocation directement sur la carte.
  • dblclick : gestion des évènements sur double-clic.
