Ce plugin a été développé à partir des plugins Gis Escoitar et Google Maps API. Certes, plus une seule ligne de subsiste, mais il serait malhonnête de prétendre être parti de rien.

Il utilise les librairies suivantes :

  • Google Maps API V2
  • Extension MarkerManager pour Google Maps API V2
  • Google Maps API V3
  • Extension Google Earth pour Google Maps API V3
  • jQuery (intensivement !)

Il s'appuie sur les plugins SPIP suivants :

  • SPIP Bonux
  • Médiathèque
