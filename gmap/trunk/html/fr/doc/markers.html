Comment sont définis les marqueurs ?

Les marqueurs sont des images au format PNG. Les marqueurs par défaut se situent dans le dossier "images" du plugin.

Points de customisation pour les marqueurs

  • Redéfinir des images brutes : le plugin utilise les fonctions de SPIP pour chercher les images avec le nom "images/nom-de-l-image.png". Si vous créez un répertoire "images" dans votre dossier squelettes et y déposez des images qui ont le m&ecric;me nom les images par défaut, les votre seront utilisées.
    Les images par défaut se nomment gmap-[marker|shadow][-objet][-full][-high].png. Le [-full] indique une icone avec l'ombre (utilisée dans les fichiers kml). Le [-high] indique une icone sélectionnée.
  • Redéfinir les fichiers de description : pour trouver le nom des images à utiliser, le plugin lit un fichier XML (selon le schéma geomarkers.xsd) qui indique les images et les informations supplémentaires nécessaires pour les utiliser sur les cartes.
