Rôle

Balise utilitaire utilisée pour récupérer depuis un squelette le chemin d'accès à une fichier qui n'est ni pas un squelette (ce que ne permet pas #URL_PAGE).

La balise utilise directement la fonction find_in_path pour renvoyer une url. Elle peut être combinée avec le filtre url_absolue pour obetnir une url absolue.

Paramètres

La balise n'utilise qu'un paramètre qui n'est pas nommé. Ce paramètre est un nom de fichier avec ou sans chemin relatif.
