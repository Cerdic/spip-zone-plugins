Rôle

Cette balise est utilisée dans les squelettes de fichier KML (comme gmap-query-default.html) pour déclarer automatiquement le texte correspondant à un point.

Le texte est utilisé dnas les info-bulles, autant par GMap lui-même que par des clients classique de fichiers KML comme Google Earth.

Le texte est extraits des fichiers gmap-info-objet[(=|-)id_rubrique].html. Ces fichiers sont des squelettes SPIP dans lesquels on peut donc utiliser des boucles et des balises.

Le plugin GMap défini des fichiers par défaut : gmap-info-default.html, gmap-info-rubrique.html, gmap-info-article.html et gmap-info-document.html.

Paramètres

NomDescriptionValeur
id_article/id_rubrique... Objet à partir duquel les points sont comptés. Ce paramètre est habituellement pris de l'environnement de la balise (la boucle englobante). {id_article} ou {id_article=39}, {id_rubrique}, {id_document}, {id_breve}, {id_mot}, {id_auteur}
type_point Type du marqueur recherché sur un élément. Parmi les types définis dans le paramétrage du plugin.
objet_parent Type de l'objet parent, passé au squelette pour faire référence, par exemple à l'article auquel appartient un document. Nom d'un objet SPIP.
id_objet_parent Identifiant de l'objet parent, passé au squelette pour faire référence, par exemple à l'article auquel appartient un document. Identifiant numérique.
