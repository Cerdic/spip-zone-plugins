/**********************************************************/
/*                    SPIP-Listes                         */
/**********************************************************/

   - Permet de fabriquer des newsletters, manuellement ou automatiquement, à partir d'un site sous Spip (http://spip.net/)
Pour ce faire Spip-listes vous permet de :
   * rédiger des messages,
   * gérer des listes d'abonnés, chaque abonné étant libre de s'abonner ou se désabonner seul et de choisir son format de message – html ou texte,
   * envoyer des informations par  e-mail, à fréquence régulière si vous le désirez,
   * envoyer les dernières parutions du site,
   * modifier à votre convenance la mise en page de vos envois.

   - La todo list se trouve sur le site bloog.net : http://bloog.net/rubrique.php3?id_rubrique=109


/**********************************************************/
/*              Installation de SPIP-Listes               */
/**********************************************************/

I. Installation des fichiers du plugin spip-listes

1) Décompresser l'archive .zip

2) installer, par FTP le repertoire obtenu spip-listes (et ses sous-répertoires), sans changement, en conservant l'arborescence, dans le répertoire /plugins (que vous devez créer s'il n'existe pas).


II. appeler Spip-listes dans vos squelettes

1) Ajoutez la balise suivante dans un squelette fréquemment appelé (au tout début de la page d'accueil ou de l'entête du site par exemple).

#MELEUSE_CRON

Cette balise déclenche l'envoi des éventuels mails en attente.

2)  ajouter le code #FORMULAIRE_ABONNEMENT dans le squelette ou vous souhaitez afficher le formulaire d'inscription.
Vous pouvez utiliser un squelette personnel à la place de inscription_listes.html (pour gérer l'inscription à une liste particulière par exemple) en utilisant la notation #FORMULAIRE_ABONNEMENT{mon_squelette}.

Note : La page abonnement.php3/html gère des modifications/désinscriptions des abonnés.

3) Dans le repertoire du plugin, éditer les fichiers /patrons/nouveautes.html et /patrons/patron.html, et adapter les adresses des images à votre site.
Attention en cas de sélection fondées sur des boucles sur l'âge, le squelette doit retourner un message vide en cas d'erreur.
Vous pouvez créer tous les patrons que vous désirez, en les déposant dans ce répertoire, ils apparaîtront automatiquement dans les choix de la partie privée.

III. Utiliser Spip-listes, dans la partie privée

Avant tout, indiquez dans le menu Configuration, interactivité, que vous acceptez les inscriptions des auteurs ou des visiteurs.

1) Cliquez, dans le menu de configuration, sur  Gestion des plugins.

2) Sur la ligne Plugin SPIP-Listes cochez la case Activez le plugin, puis validez

3) Cliquez, dans le menu Edition, sur lettres d'information (ou directement par (http://votre_site.net/ecrire/?exec=spip_listes))

Vous êtes prêt :

4) Créez une(des) liste(s) de diffusion, puis des abonnés (avec possibilité d'import d'une liste d'abonnés).
