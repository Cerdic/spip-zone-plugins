Squelettes RERS (sous forme de plugin)


______ version 0.2 - juin 2009

Utilisation du plugin CFG pour indiquer les rubriques OFFRES et DEMANDES par exemple et utiliser ces
  données dans les squelettes.


______ version 0.1 - mai 2009



Ce plugin contient, mélangés : 
- des squelettes surchargeant le plugin Ahuntsic 1.0
- des surcharges du noyau spip 2.0.5
