- Redefinir sur un diapo les largeurs et hauteurs des parametres definis dans la page de configuration du plugin
- Possibilite d'utiliser les options du plugin jquery jcycle definies ici : http://malsup.com/jquery/cycle/options.html
- Mise a jour du plugin jquery jcycle : 2.88 (08-JUN-2010)
